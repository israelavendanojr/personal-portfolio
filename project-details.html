Guassian Filter

This project involves developing a custom Unix shell in C, named myshell, that mimics the core functionalities of a terminal. The shell supports built-in commands such as pwd, cd, and exit, while also executing external system commands. It utilizes the fork-exec-wait pattern to manage child processes efficiently.

myshell features a robust argument parsing system that processes user input, handling commands and their arguments with flexible whitespace support. The implementation ensures proper memory management through dynamic allocation, with validation using Valgrind to detect and optimize memory usage.

Key Features

  • Custom Command Execution supporting both built-in and external commands.
  • Argument Parsing with support for flexible whitespace handling.
  • Process Management using the fork-exec-wait pattern.
  • Memory Management with dynamic allocation and Valgrind optimization.
  • Performance Optimization by identifying optimal execution strategies.

Unix MicroShell

Custom Unix Shell is a C-based project that involved developing a microshell, myshell, which mimics the functionality of a terminal. The shell supports both built-in commands (e.g., pwd, cd, exit) and external system commands, allowing the user to execute typical terminal operations.

The program reads user input line-by-line, parses the commands, and executes them either as built-in functions or default Unix commands.

The project also focused on dynamic memory allocation and the use of Valgrind to detect and optimize memory leaks, ensuring efficient memory management throughout the program. Overall, the project demonstrated a deep understanding of system calls, memory management, and process execution in C while emphasizing robust code quality and optimization.

Key Features

  • Custom argument parsing and input handling, which allow for flexible whitespace management and seamless execution of commands and their arguments.
  • Edge Detection using a 3x3 Laplacian Filter applied via convolution.
  • Performance Measurement by tracking execution time.
  • Automated Processing via a Bash script.
